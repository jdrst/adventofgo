`golangci-lint run` (see [github.com/golangci/golangci-lint](https://github.com/golangci/golangci-lint))

### within day
`go test`

### Benchmark:
`go test -bench=B -benchmem`

### all of them (in project root)
`go test ./...`

## Run:

### within day
`go run .`
